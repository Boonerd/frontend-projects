# NovaBoost — Marketing Automation Landing Page

A **clean, responsive, modern** landing page for **NovaBoost**, a simple and affordable marketing automation tool for small teams.

---

## Features

- **Fully Responsive** – Works on mobile, tablet, and desktop  
- **Dark Mode Toggle** – Moon/Sun icons with smooth transition  
- **Mobile Hamburger Menu** – Slide-in navigation  
- **3 Pricing Plans** – Starter, Pro (Most Popular), Enterprise  
- **Real Checkmark Icons** – No text, just checkmarks  
- **Modern UI** – Gradient text, micro-interactions, card hovers  
- **SEO Optimized** – Meta tags, semantic HTML, fast loading  
- **Zero Dependencies** – Pure HTML, CSS, JS  

---

## Tech Stack

- **HTML5**  
- **CSS3** (Flexbox, Grid, CSS Variables)  
- **Vanilla JavaScript**  
- **Google Fonts** – Inter  
- **Font Awesome** – Icons

## How to Use

1. **Clone or download** the project.  
2. Open `index.html` in your browser.  
3. Click the **Moon/Sun** icon to toggle dark mode.  
4. On mobile: tap the **hamburger menu** to open navigation.

> No build tools required. Ready to use.

---

## Customization

- **Colors:** Edit `:root` variables in `style.css`.  
- **Pricing:** Update `<section id="pricing">` in `index.html`.  
- **Features:** Update `<section id="features">` in `index.html`.  
- **Logo / Text:** Edit `<h1>` or `.logo` in `index.html`.
